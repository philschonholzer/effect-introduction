# effect-introduction

Setup:

```bash
direnv allow
```

To install dependencies:

```bash
bun install
```

To run an example:

```bash
bun ./01-simple
```

To run the test:

```bash
bun test
```

This project was created using `bun init` in bun v1.1.2. [Bun](https://bun.sh) is a fast all-in-one JavaScript runtime.
